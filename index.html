#1 : BIOS

BIOS stand for Basic Input Output System. Set of routines that provides basic io to the system like taking and outputing characters, etc. Modern BIOS can vary depending on the complexity of system ranging between 16mb to 32. Older systems had much smaller BIOS ranging 128kb to 1mb.
